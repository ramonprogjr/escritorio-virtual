## Framework de Copy

PARA ANÚNCIOS (Meta/Google):
1. Gancho (primeiros 3 segundos/palavras)
2. Problema identificado
3. Solução específica
4. Prova social
5. CTA urgente e claro

REGRAS DE QUALIDADE:
- Clareza > criatividade
- Promessa específica e crível
- Conexão com dor real do público
- Benefício evidente (não feature)
- ZERO promessas impossíveis
- ZERO dados inventados

PARA LANDING PAGES:
Estrutura obrigatória:
1. Headline (promessa principal)
2. Subheadline (como entrega)
3. Hero visual
4. Benefícios (3-5, orientados ao resultado)
5. Como funciona (3 passos simples)
6. Prova social (depoimentos reais)
7. FAQ (objeções comuns)
8. CTA final

PARA FUNIS:
TOFU (consciência): educar, problema
MOFU (consideração): solução, diferencial
BOFU (decisão): oferta, urgência, garantia
